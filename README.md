## 作者信息 {#项目信息}

**作者：吴承恩（中国明代杰出的小说家）**

**吴承恩 约1500年-1582年 ，男，字汝忠，号射阳山人。汉族，淮安府山阳县（今江苏省淮安市淮安区）人。祖籍安徽桐城高甸，以祖先聚居桐城高甸，故称高甸吴氏**

**中国明代杰出的小说家，是四大名著之一《西游记》的作者。自幼敏慧，博览群书，尤喜爱神话故事。在科举中屡遭挫折，嘉靖中补贡生。嘉靖四十五年 1566年 任浙江长兴县丞。由于宦途困顿，晚年绝意仕进，闭门著述。**
